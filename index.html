Mr. CSV Transformer

I will convert your Excel / comma- or tab-delimited data into one of several web-friendly formats, including HTML, JSON and XML. Data structures in languages including Python, Ruby, and PHP are available. Lastly, database insertion statements for MySQL and mongoDB can be generated. To include only certain columns, add "?targets=[names]" to the end of the URL; [names] is the column names separated by commas (no quotes, no brackets) in the order in which you wish them to appear to the URL above and refresh the page.
